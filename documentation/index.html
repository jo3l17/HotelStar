File Structure

  • images: Used to store the all images
  • css: Used to store the custome css styles, font-awesome, bootstrap, owl.carousel, Flex Slider, Magnific Popup
  • js: Used to store the custome javascripts, font-awesome, bootstrap, owl.carousel, Flex Slider, Magnific Popup
  • fonts: Used to store font-awesome fonts
  • email: Used to store email process PHP files

HTML Structure

This theme is a bootstrap based grid layout with single and multiple columns. All of the information within the page content area is nested within sections and div. The general template structure is the same throughout the template.
